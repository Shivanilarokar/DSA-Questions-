Navigate to the desired challenge folder and run the code examples provided. For instance, to calculate the Fibonacci sequence:

```python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(5))  # Output: 5
```
