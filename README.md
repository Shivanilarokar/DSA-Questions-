### Example
Here is a simple example demonstrating a Fibonacci sequence solution in Python:

```python
def fibonacci(n):
    if n <= 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# Example usage
print(fibonacci(5))  # Output: 5
