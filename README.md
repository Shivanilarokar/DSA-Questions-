### Example
Here’s how you can use the `two_sum` function:
```python
def two_sum(nums, target):
    # Example input
    # nums = [2, 7, 11, 15]
    # target = 9
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
```
