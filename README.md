## Example
Here is a simple example of how you can use the DSA Questions repository:

```python
# Example of a coding challenge
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))  # Output: 55
```
