## Example
Here is an example of how to use the Fibonacci function:

```python
def fibonacci(n):
    if n <= 1:
        return 1
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

result = fibonacci(5)
print(result)  # Output: 8
