## Usage
You can use the implemented algorithms directly from the repository. Below is an example of the Fibonacci function:

```python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# Example usage
print(fibonacci(10))  # Output: 55
```
