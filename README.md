## Example
Here is a simple example of how to use the solutions provided in this repository:
```python
# Example of a simple DSA question solution in Python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(5))  # Output: 5
```
